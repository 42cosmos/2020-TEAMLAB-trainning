- Flexbox(Flexible box)와 CSS Grid의 차이와 장단점은 무엇일까요?
	- Flexbox : 가로나 세로의 한 축 (1차원)을 기준으로 요소를 배열해 하위 아이템 정렬 용이 - 소규모 레이아웃에 적합
	- Grid : 가로세로 2차원 매트리스 기준으로 요소를 배치 - grid area를 만들어 내부 아이템 크기 및 위치 조정
